# Task Manager API

Simple Task Management REST API built with Spring Boot.

## Tech Stack
- Java 17**
- Spring Boot 4.0.2
- Spring MVC
- Spring Data JPA
- Thymeleaf
- H2 Database
- Maven


## Features
- Create task
- Get all tasks

## API Endpoints
- GET /api/tasks
- POST /api/tasks/ Get all tasks
- POST /api/tasks/Create new task (JSON)

## How to run
```bash
./mvnw spring-boot:run
